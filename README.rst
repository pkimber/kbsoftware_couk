Development
===========

Virtual Environment
-------------------

::

  virtualenv --python=python3 venv-kbsoftware_couk
  source venv-kbsoftware_couk/bin/activate

  pip install -r requirements/local.txt

Testing
-------

::

  find . -name '*.pyc' -delete
  py.test -x

Usage
-----

::

  ./init_dev.sh

Browse to http://localhost:8000/::

  user          staff
  password      letmein
